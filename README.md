# Hide Character Model in Character Info

When you open your Character Info panel, the 3D character model is automatically hidden.

## Install

Install directly from [CurseForge](https://legacy.curseforge.com/wow/addons/hide-character-model-in-character-info), or download and drop the `HideCharacterModelInCharacterInfo` folder into:

```
World of Warcraft/_retail_/Interface/AddOns/
```

Then restart WoW or `/reload`.
